# 证券营业部合规巡检工具

面向证券营业部合规经理的命令行巡检工具，极简方式完成每日市场传闻与客户群舆情排查。

## 功能特性

### 1. 市场传闻与舆情排查
- 支持按营业部名称、客户经理姓名、股票简称搜索
- 返回按时间排序的风险摘要
- 重点标出"荐股承诺收益""内幕消息暗示""客户亏损维权"等敏感表达
- 按风险等级（高/中）排序展示

### 2. 群聊文本核查
- 支持拖拽聊天记录文件到命令窗口
- 支持 .txt / .log 格式，自动识别编码（UTF-8/GBK等）
- 逐条筛出可能违规或可能引发投诉的内容
- 给出位置、原句和建议处理动作
- 按发言人员统计违规情况

### 3. 合规日报生成
- 一键生成可直接粘贴到合规日志的文字
- 包含：检查范围、发现问题、处置状态、待跟进事项
- 支持保存为文件

## 使用方法

```bash
python main.py
```

按菜单提示选择功能即可。

## 敏感词分类

| 风险等级 | 分类 | 说明 |
|---------|------|------|
| 高风险 | 荐股承诺收益 | 保证收益、保本保息等 |
| 高风险 | 内幕消息暗示 | 内幕消息、内部消息、庄家等 |
| 高风险 | 客户亏损维权 | 亏损、投诉、维权、起诉等 |
| 高风险 | 代客理财 | 代操作、帮你操作、账号密码等 |
| 高风险 | 私下收费 | 收费、会员费、分成、转账等 |
| 中风险 | 不当投资建议 | 买入、卖出、全仓、目标价等 |
| 中风险 | 夸大宣传 | 翻倍、暴涨、财富自由等 |
| 中风险 | 敏感社会事件 | 上访、静坐、群体性事件等 |
